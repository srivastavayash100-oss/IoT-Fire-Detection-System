# IoT-Based Farmer Field Fire Detection System
**[3rd-Year Academic Project | Group of 2]**

### 📌 Project Overview
An IoT-integrated system built to detect agricultural fires early and deliver emergency alerts without relying on internet connectivity.

### 💡 Technical Contributions
* **Multi-Sensor Fusion:** Developed an algorithm to correlate data from **IR Flame**, **Smoke (MQ-2)**, and **Temperature (DHT22)** sensors to minimize false positives.
* **Resilient Communication:** Integrated **GSM (SIM800L)** and **GPS (NEO-6M)** modules to send offline SMS alerts containing the precise location coordinates.
* **Environmental Calibration:** Conducted extensive field testing to calibrate sensor sensitivity under varying ambient light and temperature conditions to ensure reliable operation in outdoor rural environments.

### 🛠️ Tech Stack
* **Microcontroller:** Arduino
* **Connectivity:** GSM (SIM800L), GPS (NEO-6M), 433MHz RF
* **Sensors:** IR Flame Sensor, MQ-2 (Smoke), DHT22 (Temperature & Humidity)
